# Fonts

Download Noto Sans font files from [Google Fonts](https://fonts.google.com/noto) and place them here.

## Required Fonts:

- **NotoSans-Regular.ttf** - For Latin scripts (English, Spanish, Tagalog, Vietnamese, Mongolian)
- **NotoSans-Bold.ttf** - Bold variant for Latin scripts
- **NotoSans-SemiBold.ttf** - SemiBold variant for Latin scripts
- **NotoSansSC-Regular.otf** - For Simplified Chinese (Mandarin)
- **NotoSansTC-Regular.otf** - For Traditional Chinese (Cantonese)
- **NotoSansDevanagari-Regular.ttf** - For Hindi
- **NotoSansGurmukhi-Regular.ttf** - For Punjabi

Once downloaded, uncomment the font imports in `i18n/fonts.ts`.
